# musicplayer-by-react

### 基于React的音乐播放器.

![final effect](http://preview.ibb.co/gLbeHn/player.png)


---

### 使用

1. Clone

    将项目clone到本地

2. 安装依赖

    gitbash下cd进入目录，输入

    `npm install`

    安装项目依赖项
    
    （部分包可能需要更新
    
    `npm update`
    
    更新依赖项）

3. 执行

    `npm start`

    在本地浏览器打开 http://localhost:3000/ 查看结果。
